Setup, Install & Backup

Add charts from anywhere

Tap ＋ Add Charts (or Import on the rail) for four ways in:

• Import File — opens Android's picker, which reaches Downloads, Google Drive, Files, and Photos. So anything you save from an email attachment or a web page shows up here. Works for PDFs and images.
• Take Photo — snap a paper chart or whiteboard on the spot; it becomes a chart instantly.
• Paste Image — screenshot anything (lyrics on a webpage, a tab site, a text), then paste it in. You can also just paste anywhere in the app.
• From a Link — direct PDF/image URLs. Many sites block direct grabbing for security; if so, open it in Chrome, save or screenshot it, then Import.

Multiple photos of one song can be combined into a single multi-page chart (it asks when you import more than one image). You can also drag a file straight onto the window.

Install on your tablet

1. Open this file in Chrome on the tablet.
2. Chrome menu ⋮ → Add to Home screen (or Install app).
3. Launch it from the home screen icon — it opens full-screen with no browser bars.

If you don't see an install option: Chrome menu → make sure you're not in Incognito, then reload once. You can always run it straight from the file too.

Foot pedal (KYXAIN JSP01)

Pair it once in Android Settings → Bluetooth (it shows up as a keyboard). Open Pedal here to test/calibrate. No app pairing needed.

Gig Mode quick guide

• Load a setlist → tap a song → it opens on stage.
• Right pedal / right tap / → = next page; auto-rolls into the next song after the last page.
• Left pedal / left tap / ← = previous page; rolls back to the previous song.
• Hit Lock before you play — hides edit buttons, enlarges controls, blocks accidents. Long-press the app title or the Lock badge to exit.
• Turn on Wake so the screen never sleeps.

Shapes the sound of the Player (your backing / click track). Move a band while a track is playing and you'll hear it change live.

Load and play a track in the Player first, then the EQ will affect it.

Metronome

120BPM

The BPM follows whichever song you're on (set it in the song's edit screen). Tap “Tap Tempo” in time with the music to set it by feel. Sound needs one tap to start (phones block auto-audio).
